# STEERING DOCUMENT: DIRETRIZES ESTRATÉGICAS

## Visão Geral
Atuar como cientista de dados hospitalar para desenvolver um modelo preditivo que auxilie no diagnóstico da obesidade, tratando-a como uma condição multifatorial.

## Pilares Estratégicos
1. Foco no Diagnóstico Multifatorial: O modelo deve integrar fatores genéticos, ambientais e comportamentais para oferecer suporte real à equipe médica.
2. Qualidade Preditiva e Técnica: Priorizar um pipeline de Machine Learning robusto, garantindo que a meta de 75% de acurácia seja superada.
3. Comunicação e Valor de Negócio: O dashboard analítico deve traduzir métricas técnicas em informações úteis para a tomada de decisão no hospital.
4. Reprodutibilidade: Garantir que todo o código esteja disponível e organizado no GitHub para avaliação técnica e auditoria.

## Critérios de Sucesso
* Processo de construção do projeto é dividido em fases, sendo que cada fase deve ter o seu código independente, limpo, organizado e documentado.
* Modelo implantado e funcional via Streamlit com interface intuitiva.
* Script para um vídeo de apresentação estratégico, cobrindo a metodologia e o sistema. (Descrito em um documento tipo markdown independente e exclusivo para isso)
* Repositório GitHub profissional com documentação completa e clara.
