# REQUISITOS DO SISTEMA: DIAGNÓSTICO PREDITIVO DE OBESIDADE

## REQUISITOS FUNCIONAIS (RF)
* RF01 - Predição de Nível de Obesidade: O sistema deve prever se uma pessoa pode ter obesidade com base em seus dados.
* RF02 - Sistema de Apoio à Decisão: A ferramenta deve auxiliar médicos e médicas no diagnóstico.
* RF03 - Dashboard de Insights: O sistema deve conter uma visão analítica com os principais insights obtidos no estudo.
* RF04 - Apresentação de Negócio: A interface e os dados devem ser apresentados sob uma ótica de valor para o ambiente hospitalar.

## REQUISITOS TÉCNICOS (RT)
* RT01 - Performance Mínima: O modelo de Machine Learning deve possuir assertividade (acurácia) superior a 75%.
* RT02 - Deploy em Produção: A aplicação deve ser desenvolvida obrigatoriamente utilizando Streamlit.
* RT03 - Pipeline de Dados: O código deve demonstrar de forma clara as etapas de feature engineering e treinamento do modelo.
* RT04 - Stack Tecnológica: Utilização de Python, Scikit-learn, Pandas, Numpy, Matplotlib e Seaborn.
* RT05 - Documentação e Vídeo: Entrega de links funcionais e vídeo de apresentação de 4 a 10 minutos.
